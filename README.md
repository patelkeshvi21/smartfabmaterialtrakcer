# SmartFab Material Tracking & Costing App

## Project Overview
A comprehensive Flutter-based mobile application designed for SmartFab Industries to streamline material tracking, cost calculation, and inventory management.

## Key Features
- Role-Based Access Control (RBAC)
- Real-Time Material Tracking
- Automated Cost Calculations
- Offline Functionality with Cloud Sync
- QR/Barcode Scanning
- Inventory Management
- Reporting & Export Capabilities

## Technical Stack
- Flutter
- Firebase (Authentication, Firestore)
- Provider for State Management
- Hive for Local Storage
- Mobile Scanner for Barcode/QR Scanning

## Setup Instructions
1. Clone the repository
2. Run `flutter pub get`
3. Configure Firebase credentials
4. Run the app with `flutter run`

## Security Notes
- Sensitive configurations are gitignored
- Role-based access prevents unauthorized data modifications

## Development Environment
- Flutter SDK: Latest Stable Version
- Dart SDK: Compatible Version
- Firebase Project: SmartFab Material Tracker
